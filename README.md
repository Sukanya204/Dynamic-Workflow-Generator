# 🚀 Dynamic Workflow Generator

A modern, responsive **AI-inspired workflow generator** built using **HTML, Tailwind CSS, and Vanilla JavaScript**. Describe a business process in plain English, and the application generates a visual workflow that you can edit, customize, and export.

---

## 📖 Overview

Dynamic Workflow Generator converts natural language descriptions into structured workflow steps. It simulates an AI-powered workflow builder by generating common business process flows, allowing users to modify them through an intuitive interface.

---

## ✨ Features

- 🤖 Generate workflows from natural language prompts
- ✏️ Edit workflow step names and types
- ➕ Add custom workflow steps
- ❌ Delete workflow steps
- 🔄 Reset the workflow
- 📊 Live step counter
- 📥 Download workflow as a `.txt` file
- 📱 Fully responsive design
- 🎨 Clean modern UI with Tailwind CSS
- ⚡ Fast and lightweight (No frameworks required)

## 📝 Workflow Step Types

The editor currently supports the following step categories:

- Input
- Action
- Approval
- Decision
- Notification
- End

Each step type is represented with its own icon and badge.

---

## 📥 Export

The workflow can be exported as a readable text file.

Example:

```text
========================================
DYNAMIC WORKFLOW EXPORT
========================================

Generated: 07/07/2026

Total steps: 5

--- WORKFLOW STEPS ---

1. [Input] HR collects documents
      ↓
2. [Action] IT allocates laptop
      ↓
3. [Action] Manager schedules induction
      ↓
4. [Action] Employee completes training
      ↓
5. [End] Onboarding Complete

========================================
```

## ⚙️ How It Works

1. Enter a business process in natural language.
2. Click **Generate**.
3. The application simulates an AI response and creates workflow steps.
4. Edit or customize the generated workflow.
5. Add or remove steps as needed.
6. Export the workflow as a text file.
